# Компоненты системы

## Основные модули

- `app/ui/` — PyQt5-интерфейс: окна, формы, таблицы отчетов, сценарии оптимизации и развертывания.
- `app/core/database/` — SQLAlchemy-модели, инициализация БД, сиды, миграции совместимости.
- `app/core/repositories/` — CRUD-слой доступа к данным.
- `app/core/services/device_service/` — подключение к устройствам (Android/Linux/Raspberry Pi), сбор характеристик, OTA-деплой.
- `app/core/services/optimization_service/` — оптимизация моделей ONNX/TensorFlow, квантизация, конвертация PyTorch -> ONNX.
- `app/core/services/validation_service/` — расчет метрик качества до/после оптимизации.

## Поддерживаемые форматы

- Исходные модели: `h5`, `keras`, `onnx`, `pb`, `tflite`, `pt`, `pth`.
- Оптимизированные артефакты:
  - ONNX-пайплайн: `onnx`
  - TensorFlow-пайплайн: `tflite`

## Поток данных

1. Пользователь добавляет исходную модель.
2. Пользователь добавляет устройство и сохраняет его доступы SSH в БД.
3. Оптимизация формирует артефакт под выбранный профиль.
4. Валидация считает метрики качества.
5. Деплой отправляет оптимизированный артефакт на целевое устройство.
