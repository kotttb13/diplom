# Развертывание и эксплуатация

## 1) Установка на Windows

1. Установите Python 3.10+ (с опцией `Add Python to PATH`).
2. В корне проекта запустите:
   - `setup.bat`
3. После установки:
   - `.venv\Scripts\activate.bat`
   - `python app/main.py`



## 2) Подготовка устройства

Для любого устройства должны быть доступны:
- SSH-подключение (IP/порт/логин/пароль),
- свободное место в целевой директории деплоя.

### Android (Termux) пример

```bash
pkg update
pkg install openssh python
whoami
passwd
sshd
```

## 3) Работа в приложении

1. Добавьте модель (`Модели`).
2. Добавьте устройство (`Устройства`) и сохраните SSH-учетные данные.
3. Выполните оптимизацию (`Оптимизация`).
4. Выполните развертывание (`Развертывание`).

## 4) Docker-лаборатория (опционально)

Контейнеры нужны только для локального теста SSH/деплоя.  
Это **не обязательный** шаг и не часть `setup.bat`.

```bash
docker-compose up -d --build
docker-compose down -v
```

## 5) Диагностика

- Если не проходит проверка связи — проверьте, что у устройства в БД сохранены `username/password/port`.
- Если деплой не проходит — проверьте SSH-доступ и права в `remote_dir`.
- Если не ставятся зависимости — проверьте доступность зеркал из `setup.bat`.
